HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • The title element defines the title for the webpage
  • The style element defines the CSS styles associated with the website
  • The base element defines the base URL for the webpage
  • The link element connects an external resource to the HTML document

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much sapce we want around the content inside an element.
  • An asterisk declares all the elements on the page will have the same style properties

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds changes from the current working branch to the staging area to be commited
  • git commit -m "commit message": captures a snapshot of the current staged changes and the branch is updated with the commit message
  • git pull origin main: receiving the modifications of the main branch into the local environment from the GitHub repository
  • git push origin "name of branch": updates changes we have locally to the remote GitHub branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A string is a sequence of text that is enclosed within quotation marks.
  • A boolean is a true or false value
  • An array is a structure which allows you to store multiple values in a single reference
